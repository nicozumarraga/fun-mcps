# Kanye West Quotes MCP Server

An MCP server that provides random Kanye West quotes.

## Usage

### With npx

```bash
npx kanye-mcp
```

### With Claude for desktop

Add to your Claude for Desktop configuration:

```json
{
  "mcpServers": {
    "kanye": {
      "command": "npx",
      "args": ["-y", "kanye-mcp"]
    }
  }
}
```

## Features

* *   Get random Kanye West quotes
